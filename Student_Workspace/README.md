# Uniswap V3 底层状态机仿真器

## 项目概述

本项目实现了一个 Uniswap V3 底层状态机的仿真器，用于模拟去中心化交易所的核心物理行为。作为量化对冲基金的 CTO，您的任务是构建一个坚不可摧的沙盒环境，为未来的 AI Agent 提供真实的交易模拟环境。

### ✅ 已实现功能

1. **池子初始化 (Pool Initialization)**
   - 支持设定初始价格 `sqrtPriceX96` 和初始流动性 `L`
   - 实现 Q64.96 定点数格式的价格转换

4. **高级测试与分析功能**

   - 价格波动场景压力测试
   - 极端市场条件下的系统稳定性验证
   - 双向交易跨区间验证

5. **核心数学红线验证**
   - 零头寸交易不影响池子状态
   - 交易后 X × Y ≥ K 的不变性验证
   - 极端大单的优雅处理

## 核心数学原理

### Q64.96 定点数表示法

Uniswap V3 使用 Q64.96 定点数格式来避免浮点数精度问题：

```python
# 魔法常数 2^96
Q96 = 2**96

# 价格转换公式
sqrtPriceX96 = sqrt(P) × 2^96
P = (sqrtPriceX96 / 2^96)^2
```

### 交易数学公式

**用 token0 购买 token1：**
```
Δy = L × (√P - √P')
Δx = L × (1/√P' - 1/√P)
```

## 决策备忘录

### LP 死亡滑点分析

在模拟了 V3 机制后，作为 LP 的主要风险在于：

1. **流动性分布不当导致的闲置资本**：如果价格区间设置过宽，大部分流动性在价格波动时无法被利用
2. **无常损失放大效应**：V3 的集中流动性机制在价格大幅波动时可能导致比 V2 更大的无常损失
3. **区间边界滑点风险**：当价格接近流动性区间边界时，交易滑点急剧增加

**关键洞察**：成功的 V3 LP 策略需要精准的价格预测和动态的区间调整，否则资本效率可能反而低于传统的 V2 模式。
