# 🎯 Task 00：环境配置与破冰 (Environment Setup & Onboarding)

> **发布日期：** 2026 春季学期  
> **作业性质：** 个人作业 (Individual)  
> **提交通道：** GitHub Classroom 专属私有仓库  

## 📍 任务目标 (Objective)
这是本课程的“第 0 次课设”。它的核心目的不是考核建模能力，而是确保每一位同学都能**打通硅谷级代码协作平台（GitHub）的全维工作链路**，扫清后续正规军团作战时的基建障碍。

当你能够完整阅读到这份文件时，说明你已经成功突破了前两层结界（注册账号 + 接受任务创建了私人克隆仓库）。

## 📝 你的通关试炼 (The Deliverable)

为了证明你已经完全掌握了“修改代码 -> 暂存 -> 提交 -> 推送云端”的**「三段式咒语」**，你需要完成以下一次极简的模拟作业提交：

1. **进入专属实验台：**
   在你的电脑本地，进入你刚克隆下来的作业文件夹中，找到名为 `Student_Workspace` 的目录。

2. **创建你的第一份实验报告：**
   在 `Student_Workspace` 文件夹内，新建一个以你的名字命名的 Markdown 文件。
   例如：`张大山_Task00_破冰实验.md`。

3. **写入破冰铭文：**
   使用任何你喜欢的文本编辑器（如 VS Code, Typora 或直接在这个文件里写入），随意写下几句话。比如简单介绍一下你自己，或者写下一句：
   > “Hello World! 我的本地环境配置与 GitHub 通信链路已完全打通。”

4. **念动提交咒语 (最关键的一步！)：**
   打开终端 (Terminal / Git Bash)，确保路径在你的作业主目录里，依次执行：
   ```bash
   git add .
   git commit -m "feat(task00): 提交破冰作业和个人介绍"
   git push
   ```

## 🚥 如何确认通关？ (Verification)
执行完这三句咒语后，请打开浏览器，回到你这份作业所属的 GitHub 个人仓库页面（也就是 `https://github.com/.../Task00-环境配置与破冰-[你的名字]`）。

如果你能在网页上的 `Student_Workspace/` 文件夹里，**看到你刚才新建的那个文件和你写下的那句话**，恭喜你——

🎉 **你已经彻底征服了本课程的核心基础设施，开启你的建模之旅吧！**
