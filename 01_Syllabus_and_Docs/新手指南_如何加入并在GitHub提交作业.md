# 🚀 新手村历险记：如何进入 GitHub Classroom 作业空间 (Onboarding Guide)

欢迎同学们来到《管理建模与仿真》这门课的云端战场！

在这门课中，我们不再使用传统的邮件或学习通提交代码作业。我们将完全模拟 **硅谷级科技公司的真实工业环境**，使用全世界最强大的代码协作平台 —— GitHub 来管理你们所有的实验、代码和报告。

请务必按照以下步骤，在 **第一次上课前** 完成你的账号绑定与结界穿越。

---

## 🛑 第一层挑战：注册 GitHub 账号 (The Identity)
如果你还没有任何 GitHub 账号：
1. 访问地球上最大的开源社区：**[https://github.com/signup](https://github.com/signup)**。
2. 输入你的常用邮箱并按照炫酷的星际动画指引，完成账号注册。
3. 请记住你的用户名和密码。

---

## 🌟 第二层挑战：穿越结界绑定课堂 (The Portkey)
每当全知的主讲老师发布了一次全新的周作业或实验课设，他都会在群里丢出一条长得很像下面这样的神秘链接：

> 🔗 **课程作业接受入口（示例）：**
> `https://classroom.github.com/a/神秘代码_Xyz`

**你要做的神圣仪式：**
1. **点击这条邀请链接！**
   *(系统会要求你先登录前面注册的那个 GitHub 账号)*。
2. **身份认领大厅 (Roster Matching - 如果有的话)：**
   你可能会看到一个包含了全班同学长名单的页面。请在浩瀚的名单中**找到你现实世界里的真实学号/姓名**，并点击它。
   *(注意⚠️：这一步只有第一次进课才需要，一旦认领，你的网络虚拟人设将永久与学号锁定。请千万不要手滑点成别人，否则你一整个学期都在给别人交卷子！)*
3. **接受任务协议：**
   点选页面中央显眼的绿色大按钮：**【Accept this assignment】** (接受此作业挑战)。

---

## 📦 第三层挑战：唤醒你的专属隐形仓库 (The Clone)

按下按钮后，请稍微闭目等候几秒钟的魔法汇聚...
随后，系统会在云端的无人区，**为你单独炸开一个极其私密的“平行宇宙”子空间！**

此时页面会显示一条新的仓库链接，长这样：
`https://github.com/simu-class-2026/Week01作业-王小明`

**点击进入你的专属仓库！你会发现：**
*   里面已经自动装载了老师提供的原版课件、数据包。
*   里面有一个专门的目录叫 **`Student_Workspace/`**，这就是你的作图纸和实验台。
*   🚀 **最重要的是：这是一个【Private/私有】级权限的结界。** 意味着在这个世界上，除了你和导师，没有第三个同学或者碳基生物能看到你写了什么或抄走你的作业。

---

## 💻 终极绝杀：提交作业的三段式咒语 (The Submission)

不要直接在浏览器页面上改代码！请在你的电脑本地打开终端（Terminal）或 Git 客户端，开始你的炼金术：

**1. 把工作台拉取到你的电脑里：**
```bash
git clone <获取你刚才那个私密仓库的绿色Code按钮里的链接>
cd <进入该文件夹>
```

**2. 开始写作业：**
尽情地开始编码建模写 markdown 吧！把你的各种文件塞进那个 `Student_Workspace/` 文件夹即可。

**3. 保存进度并提交收工 (云端同步)：**
当你敲完代码或准备关机时，只要向控制台输入这三个被后世称为**“提交三段式”**的上古咒语：
```bash
# 第一句：把今天的所有改动放进传送法阵
git add .
# 第二句：给包裹贴上时间戳标签
git commit -m "报告：完成了第一问和主函数模型"
# 第三句：按下发射按钮，将进度轰回云端母体！
git push
```

只要没报错，你可以随时把刚提交的链接关掉。**导师在期末考核时，会自动从后台把你们全班最新的云端状态一键抓取下来批改。** (是的，你无须发任何消息对老师说“我交了”)。

祝你建模与代码皆无 bug！
